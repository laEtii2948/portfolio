Projet réalisé seule – Application streamlit

Introduction

Projet réalisé dans le cadre de mon cours de programmation en python. L'objectif était de développer une application web interactive avec Streamlit pour explorer et visualiser un jeu de données choisi.

Fonctionnalités

  • Affichage interactif du jeu de données choisi
  • Filtrage et sélection des données selon différents critères
  • Affichage de la charge sous forme de graphique évolutif

  • Calcul et affichage des statistiques descriptives (moyenne, médiane, max, min, etc.)

  • Visualisation graphique des données (histogrammes, courbes)
  • Interaction avec les graphiques pour explorer les valeurs
  • Mise à jour dynamique des graphiques selon les filtres appliqués

  • Filtres et menus déroulants pour sélectionner les données

Aspects techniques

  • Langage utilisé : python
  • Manipulation de données avec la librairie Pandas
  • Visualisation de données avec la librairie Matplotlib
  • Création d'interfaces web simples avec la librairies streamlit hébergées sur streamlit cloud
  • Gestion des filtres et des interactions utilisateurs avec streamlit

Difficultés rencontrées

Le principal défi de ce projet a été de concevoir une interface utilisateur cohérente et intuitive, capable de présenter les données et les graphiques de manière claire. Il a également fallu utiliser à bon escient les différentes bibliothèques Python (Pandas, Matplotlib, Streamlit) pour que l’analyse, la visualisation et l’interactivité fonctionnent ensemble de façon fluide. La préparation des données, le choix des graphiques et l’organisation du code pour qu’il reste modulable ont aussi demandé une attention particulière. Ces obstacles ont été enrichissants et m’ont permis de renforcer mes compétences en développement d’applications interactives et en visualisation de données.

Conclusion et améliorations futures

Ce projet m’a permis de mettre en pratique mes compétences en Python, en analyse de données avec Pandas, en visualisation avec Matplotlib et en création d’interfaces interactives avec Streamlit. Il m’a appris à concevoir une application cohérente, intuitive et capable de gérer différents types de données de manière dynamique. On pourrait en piste d'amélioration intégrer des données en temps réel afin de suivre l'évolution de la charge de manière fine.
